DB
- 구조화된 정보, 데이터의 조직화된 모음
- 일반적으로 컴퓨터에 전자적으로 저장이 되고 데이터베이스 관리시스템에 의해 제어된다.

DBMS(DataBaseManagermentSystem)
- 데이터베이스와 사용자 또는 프로그램 간의 매개체 역할을 하여 사용자가
- 데이터의 구성, 검색, 수정, 삭제와 같은 관리를 할 수 있게 해준다.

DBMS의 종류
- ORACLE (기업용은 유료, 교육용 무료)
- MySQL
- Microsoft Access
- Microsoft SQL Server
- MariaDB

큰 틀은 비슷비슷함

DB는 행과 열 형태로 저장이 된다.

SQL문
- 원하는 결과를 얻어오기 위해 DB에 요청하는 문장(Query문)
- SQL문장은 대소문자를 가리지 않는다.
- 한 줄 또는 여러 줄에 걸쳐서 입력하는 것이 가능하다.
- SQL 문장의 끝은 세미콜론으로 끝맺어야 한다.

SQL문장의 종류
1. DML : 데이터 조작어. 실질적으로 가장 많이 사용하는 쿼리문이 될것
- SELECT, INSERT, UPDATE, DELETE
2. DDL : 테이블 조작과 관련된 쿼리문
- CREATE, ALTER, DROP
